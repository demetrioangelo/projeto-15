# 🌺 Flores Online - Sistema de Contato

Sistema completo de formulário de contato para floricultura com validação de dados e segurança.

## 🚀 Funcionalidades

- ✅ Formulário de contato com validação
- ✅ Proteção contra ataques (XSS, SQL Injection)
- ✅ Sanitização de dados
- ✅ Armazenamento em arquivo .txt
- ✅ Interface responsiva
- ✅ Mensagens de erro amigáveis

## 🛠️ Tecnologias

- Python 3.x
- Flask
- HTML5
- CSS3

## 📦 Como instalar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/flores-online.git

# Entre na pasta
cd flores-online

# Instale as dependências
pip install flask

# Execute o servidor
python app.py
